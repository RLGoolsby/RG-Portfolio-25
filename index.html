About Me

My journey into frontend web development wasn't traditional—but it was fueled by curiosity and a passion for creativity. I was looking for a challenge that would let me build visually engaging, interactive experiences. I found that in coding.

Over the past few years, I’ve built responsive, accessible web applications using HTML, CSS, JavaScript, and jQuery—often collaborating with designers, marketers, and backend developers. I love turning complex ideas into clean, intuitive user interfaces and continually exploring new tools to create polished digital experiences.
